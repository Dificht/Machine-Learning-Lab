Этот проект использует датасет LUNA для подготовки данных, извлечения 3D‑патчей из КТ лёгких и визуализации кандидатов узлов. 
Основные части:
- notebooks/dsets.py — формирование датасета для PyTorch
- notebooks/vis.py — визуализация кандидатов
- util/util.py — преобразование координат
- util/disk.py — кэширование данных
- explore_dset.ipynb — исследование и анализ

## Установка
1. Клонировать репозиторий
2. Установить зависимости (numpy, torch, SimpleITK, matplotlib, diskcache)

## Датасет

⚠️ Внимание: сам датасет LUNA16 не включён в репозиторий, так как имеет очень большой размер (сотни гигабайт).  
Для работы с проектом необходимо скачать его отдельно.

### Скачивание
Датасет доступен на официальной странице конкурса LUNA16:  
http://luna16.grand-challenge.org/download/

После скачивания разместите данные в папке `LUNA_dataset/` в корне проекта.
